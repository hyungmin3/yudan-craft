# YUDAN CRAFT

Lightweight voxel sandbox inspired by Minecraft, built with `Three.js + TypeScript + Vite`.

## Scripts

- `npm.cmd install`
- `npm.cmd run dev`
- `npm.cmd run build`
- `npm.cmd run test`

## Features

- Seeded overworld / nether / end generation
- Survival, creative, and hardcore modes
- Block mining, placement, crafting, furnaces, and chests
- Structures including villages, school, dungeons, and strongholds
- Desktop + mobile HUD and controls
- IndexedDB world saves
